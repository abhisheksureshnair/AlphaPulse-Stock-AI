# AI Stock Trading Assistant - Flutter App

A premium, AI-powered stock trading assistant app with a modern dark theme, neon accents, and glassmorphic design elements.

## Features
- **Modern Dashboard**: High-contrast charts and AI-integrated stock watchlist.
- **AI Analysis Breakdown**: Custom sentiment gauges and model intelligence tracking.
- **Detailed Stock Insights**: Glowing price charts and AI confidence scoring.
- **Dynamic Alerts**: Signal-based trading notifications with confidence filtering.
- **Premium Profile**: User stats and AI strategy configuration.

## Getting Started

1. **Prerequisites**: Ensure you have Flutter installed on your machine.
2. **Install Dependencies**: Run the following command in the project root:
   ```bash
   flutter pub get
   ```
3. **Run the App**:
   ```bash
   flutter run
   ```

## Project Structure
- `lib/theme.dart`: Design tokens, colors, and global theme.
- `lib/main.dart`: App entry point and navigation.
- `lib/screens/`: Individual screen implementations.
- `pubspec.yaml`: Project dependencies (Google Fonts, FL Chart, etc.).
